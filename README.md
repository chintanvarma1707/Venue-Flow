> **The ultimate Digital Twin platform for high-performance stadium management and premium fan engagement.**

ApexStadium transforms traditional sports venues into living, data-driven ecosystems. Featuring a tactical **"God Mode"** Admin HUD, holographic seat mapping, and neural Vision AI, it provides total venue oversight in a stunning glassmorphism interface.

---

## ✨ Key Features

### 🔵 Fan Experience Hub
*   **Holographic Seat Map**: Interactive 3D stand visualization with seat-level availability and real-time reservation.
*   **Neural Vision AI**: Live crowd density heatmaps powered by simulated neural tracking nodes.
*   **Smart Concessions**: Seamless in-seat food ordering system with real-time wait-time prediction.
*   **Tactical Emergency SOS**: High-visibility silent alert system for immediate security dispatch.

### 🛡️ Administrative Command Center (God Mode)
*   **Global Digital Twin**: A massive central visualization of the stadium health, density, and flow.
*   **Neural AI Feed**: Monitor live gate occupancy and wait times via synchronized data nodes.
*   **Live Simulation Engine**: Stress-test your venue with dynamic crowd-flow simulations that update in real-time across all fan devices.
*   **Property Overrides**: Manually redirect traffic flows and gate assignments during peak-load anomalies.

---

## 🎨 Modern UI/UX Design System

ApexStadium uses a custom-crafted **Tactical Glassmorphism** design system:
- **Rich Aesthetics**: Deep slates, neon cyans, and vibrant accent colors (Critical Red, Optimal Green).
- **Micro-Animations**: Fluid transitions powered by `framer-motion` for every modal, HUD, and data point.
- **Responsive Core**: Auto-adjusting bento-grid layouts that scale perfectly from 4K monitors to mobile devices.
- **Interactive Layers**: Multi-dimensional SVG maps with hover-aware telemetry.

## 🏗️ Professional Architecture

The project follows a modular, industry-standard directory hierarchy for maximum scalability:

```text
src/
├── app/               # Next.js App Router & API Bridges
├── components/
│   ├── admin/         # Tactical Command & Super Admin controls
│   ├── auth/          # Onboarding & Authentication flows
│   ├── features/      # Live features (VisionHub, FanSync, Concessions)
│   ├── stadium/       # Visualizers (StadiumBowl, SeatMap, Heatmap)
│   └── ui/            # Design system primitives (Navs, Cards, Modals)
├── lib/               # Centralized data models & constants
└── types/             # Global TypeScript definitions
```

## 🚀 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/chintanvarma1707/Venue-Flow.git
cd ApexStadium
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run Development Server
```bash
npm run dev
```

### 4. Admin Access
Navigate to `/admin` and use the following credentials:
- **Access ID**: `admin`
- **Security Key**: `password`
